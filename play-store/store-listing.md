## Full Description

### English
Animal Merge Game is a soft, cozy merging puzzle you can play with one hand.

Drop animal plushies into the play field, line up matching animals, and merge them into larger forms. The more carefully you stack, the longer your run lasts and the higher your score climbs.

Features:

- Tap-and-drag controls tuned for phones
- Japanese and English language support
- Local best-score tracking on your device
- Cute plush-style animal pieces
- Quick rounds that fit short breaks
- Plays offline once installed

A relaxing puzzle for short breaks, score chasing, and unwinding.

### Japanese
Animal Merge Game は、片手で気軽に遊べる、やさしい雰囲気のマージパズルゲームです。

動物のぬいぐるみをフィールドに落として、同じ動物どうしを重ねると、より大きな姿へ進化します。置き方を工夫するほど長く続き、高得点を狙えます。

特徴:

- 端末内に保存されるベストスコア

- すきま時間に遊びやすい短めのプレイ感

気軽に遊べて、ついスコア更新を狙いたくなるパズルゲームです。

## Data Safety Summary

This is the source of truth for the Google Play "Data safety" form. Everything below is stored on the device only; nothing is transmitted to our servers.

| Data type | Collected | Shared | Optional | Purpose |
|---|---|---|---|---|
| Best score (number) | Yes (on device) | No | No | Game functionality |
| Local leaderboard entries (id + name + score) | Yes (on device) | No | No | Game functionality |
| Player display name (max 12 chars) | Yes (on device) | No | Yes (defaults to "Guest") | Game functionality |
| Locally generated player ID (random) | Yes (on device) | No | No | Game functionality |
| Language preference (ja / en) | Yes (on device) | No | Yes | App preference |
| Sound / haptics / reduced motion toggles | Yes (on device) | No | Yes | App preference |

- Account creation: not required
- Location, camera, microphone, contacts: not requested
- Network calls: none from app code (the INTERNET permission exists only because the app is packaged from web technology)
- Third-party SDKs: none
- Children: this app does not knowingly collect data from children; it does not collect data from anyone in any case
